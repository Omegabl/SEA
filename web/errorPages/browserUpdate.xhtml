Antes de continuar...

Esta página sólo funciona con las versiones mas recientes de los navegadores Mozilla Firefox, Google Chrome, Opera y Safari. Deberás hacer unas configuraciones para poder acceder al sistema. Por favor sigue las instrucciones según tu navegador:

Mozilla Firefox

Activa la bandera llamada layout.css.grid.enabled. Para ello debes dirigirte a: about:config

Firefox Developer Edition

Esta versión de Firefox ya tiene habilitadas las características que necesita el sistema, así que no tienes que hacer nada.

Google Chrome

Activa la bandera llamada Funciones experimentales de plataforma Web. Para ello debes dirigirte a: chrome://flags/#enable-experimental-web-platform-features

Opera browser

Activa la bandera llamada Enable experimental Web Platform features. Para ello debes dirigirte a: opera://flags/#enable-experimental-web-platform-features. Aun así, no aseguramos que el sistema funcione correctamente.

Safari

Este navegador tiene habilitadas las características que necesita el sistema en Safari Technology Preview y WebKit Nightly Builds; Igualmente estas características estarán habilitadas en la versión 10. Aun así, no aseguramos que el sistema funcione correctamente.

Internet Explorer

Lamentablemente no ofrecemos soporte para Internet Explorer.

Microsoft Edge

Lamentablemente Microsoft Edge no tiene soporte para las características que usa el sistema.
